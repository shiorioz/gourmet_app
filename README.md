## アプリケーション機能

### 機能一覧

- レストラン検索：ホットペッパーグルメサーチ API を使用して、現在地周辺の飲食店を検索する。
- レストラン情報取得：ホットペッパーグルメサーチ API を使用して、飲食店の詳細情報を取得する。
<!-- - 電話アプリ連携：飲食店の電話番号を電話アプリに連携する。 -->

### 画面一覧

- 検索画面 ：条件を指定してレストランを検索する。
- 一覧画面 ：検索結果の飲食店を一覧表示する。
- 詳細画面 ：飲食店の詳細を表示する。
